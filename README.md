# _Tgbot - телеграм бот_ 
____

### _Описание проекта_

Данный телеграм бот написан для работы с API. Проверяет статус выполненной работы и исходя из изменения статуса работы присылает в телеграм обновленный статус.

### _Технологии_

- Python 
- API
